# Privacy Policy

## Effective as of December 1st, 2025

This policy explains how "Yomu Reader - Read Japanese" mobile application collects, stores, uses, transfers, and shares Personal Data from its users.

## Data We Collect about You

### Personal Data

We don't collect any personally identifiable information.

### Usage Data

When you use the app, we collect anonymous statistics about your use of the app:

- Frequency of use
- Features of the app that you use

We use Journey3: Anonymous Mobile Analytics Platform (https://journey3.net/), which requires storing temporary data related to the current session on your device, in the directory for application support files.

## Handling the Books

To provide simplified versions of book content, we pre-process the books you import into the "Yomu Reader - Read Japanese" app. The books are converted into an internal format by a specialized service. For this purpose, we send the books to a private server, where they are temporarily stored in order to be processed.

- We do not track which books you import into the app
- We do not keep uploaded books any longer than necessary for the app to function properly

## Security and Your Rights

We take reasonable measures to protect data during processing.

Since we do not collect personal data, there is no personal data to access or delete.
